fold [src]
fn fold(xs: List, init: Int, f: fn(Int, Int) -> Int) -> Int

Reduce a list from left to right. Applies f to the accumulator and each element, threading the result through.

example
List.fold(Cons(1, Cons(2, Nil)), 0, |acc, x| -> acc + x)
  -- => 3

fn with_stdout_logger(_ f: () -[Log, e]> T) -[IO, e]> T
